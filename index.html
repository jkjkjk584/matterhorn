과거 월례회 이력

지난 월례회의 경기 결과, 순위, 조 편성, 행운권 결과를 조회합니다.

조회 전용 과거 월례회를 선택하면 해당 월의 경기현황, 순위, 조 편성, 행운권 결과를 볼 수 있습니다. 현재 운영 화면으로 돌아가려면 상단의 현재 월례회로 돌아가기를 누르세요.

회원 추가

정회원과 Guest를 모두 DB 회원명단에 추가할 수 있습니다.

추가한 정회원/Guest는 즉시 참가 상태로 들어가며, 저장되면 다른 기기에서도 같은 회원명단을 사용합니다. 이름·성별·실력·회원구분은 관리자 화면에서 바로 수정할 수 있습니다.

현재 운영 데이터를 월별로 보관하고, 다음 달 월례회를 새로 시작합니다.

운영 중 · 참가자 점수 입력 가능
과거 월례회를 볼 때는 조회 모드로 전환됩니다. 현재 월례회 운영으로 돌아가려면 상단의 현재 월례회로 돌아가기를 누르세요.
📊

CSV 백업 · 복구

현재 불러온 월례회 데이터를 통합 CSV로 내려받고, 원본JSON이 포함된 백업 파일로 복구합니다.

백업에는 회원명부, 조편성, 대진표, 개인순위, 팀요약, 추첨결과, 원본JSON이 포함됩니다. 복구는 CSV 안의 원본JSON을 기준으로 현재 운영 데이터를 덮어씁니다.
🔑

관리자 비밀번호 변경

관리자 비밀번호는 현장 운영용이고, 마스터키는 비밀번호 분실 시 복구용입니다.

기본 관리자 비밀번호는 0000, 기본 마스터키는 matterhorn입니다. 마스터키로도 관리자모드에 진입할 수 있고, 관리자 비밀번호를 재설정할 수 있습니다.
